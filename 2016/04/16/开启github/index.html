开启github

发表于 2016-04-16   |   分类于 生活

为了搭建github博客已经把我折腾了很长时间了，在这期间查阅了许多有关配置hexo技术博客，因为第一次接触，所以配置过程中，总出一些问题，每次出问题总感觉是自己安装的时候有问题，就这样重复了好多次，给自己挖了许多坑，而又在参阅的技术资料时发现了不同版本，导致自己走了很多弯路。不过最后凭借自己的不断的钻研，搭建成功，总体比较满意，但是里面还需要后期添砖加瓦。在这非常感谢CnFeat和zipperary两位博主，两位的博客写的非常详细，是我学习的榜样。
为什么要搭建github博客？

第一，去认识github，去学习github团队协作开发方式。

第二，世界大牛都聚集于此，开阔了自己的眼界。

第三，身在技术界，要有不断去探索发现新事物的精神。

第四，它是属于自己的独立博客域名。

第五，写博客可以重新认识自己，让别人去认识了解你。

第六，写博客可以让你坚持的做一件事情，不断积累知识，建立自己的知识体系。

搭建github博客的需要做好以下心理准备

1、非常折腾，需要一定的耐心；﻿

2、也需要一定的学习能力和钻研精神；﻿

3、懂一些网页基础知识。
