一趟远的旅行

发表于 2016-04-22   |   分类于 生活   |  

人生很短，短的来不及陪伴亲人；人生又很长，因为有好多事等待着你做，我把它们都当作我们的每一次旅行。

一趟远的旅行可能是你的学业。
从小踏入校园，你便注定学习一生。在学校，你学习书本知识，做人知识；入社会，你学习工作知识，学习处世知识。学习不仅仅为了找份好工作，也不是仅仅提升自己的文化水平，它可以使我们与社会共同进步，我们可以享受到社会发展给我们带来的便利。我们不被淘汰，我们可以做更多有意义的事，可以有更好的未来。

一趟远的旅行可能是你的事业。
你入社会便与事业结下了缘，你可以兢兢业业的在自己的工作岗位上，发挥自己应有的价值，直至退休。你也可以为了更好的未来去创造属于自己的事业，路很坎坷，也很长，你去坚持，去奋斗，万一实现了呢！

一趟远的旅行可能是你的爱情。
人们常说，最高兴的事便是与爱的人有一趟远的旅行。在这爱情也是一趟远的旅行，相识，相知，相爱，便注定了一趟远的旅行，这趟旅行也不是一帆风顺的，有大风大浪的考验，也有暗礁险滩的阻挠。你们或许可以披荆斩棘，迎风远航，使得终点。你们或许因风浪暗礁而沉没。

一趟远的旅行可能是一趟真的旅行。
和爱的人有一场说走就走的旅行，可以享受一下生活的乐趣，可以体验一下大自然，可以接触一下异国风情。这趟旅行不再距离远近，不再去的地方好坏，而是你们带着什么心情去的，又带着何种心情回来的。

人生漫长，远的旅行很多，希望在每次远的旅行中，都有所收获，正如我们所期待那样。

开始调试

vsc官方文档调试链接

  1. 必须以项目的方式打开要调试的js脚本所在的目录，目录路径最好不要包含中文，否则可能有问题
  2. 在要调试的脚本文件中，找到具体要设置断点的行，在左侧点击设置断点
  3. 设置好断点之后，按F5启动调试
  4. 这个时候，vsc 编辑器会提示你选择要调试的环境，这里选择 Node.js 即可
  5. 当选择完调试环境之后，vsc 默认会在当前根目录下生成一个 .vsccode 目录
  6. 在 .vsccode 目录下找到一个叫做 launch.json 的文件，打开编辑
  7. 在 launch.json 文件中，在 configurations 节点下找到一个叫做 program 的属性节点
  8. 将 program 属性节点中原来的值 ${workspaceRoot}/app.js 改为 ${workspaceRoot}/要调试的脚本文件名.js
  9. 修改完毕之后，按 Ctrl+F5 保存
  10. 上述操作完成之后，按 F5 启动调试
  11. 尽情的享受 vsc 调试带给你的调试的乐趣吧
  12. 快捷键和 Chrome 浏览器中的调试环境快捷键一致

对于执行了以上操作还没有成功的同学，建议将 vsc 关闭重新打开再次按 F5 启动调试即可解决

1.5 webstorm 调试 node

开始调试

  1. 在要调试的文件中具体的行位置左边通过 Ctrl+F8 设置一个端点
  2. 在当前要调试的文件中通过鼠标右键，然后选择 Debug 要调试的文件名.js
  3. 这个时候 ws 会自动帮你启动调试模式，并且自动停留在你打击端点的位置
  4. F8 步进
  5. Shift + F8 步出
  6. Alt + F8 可以执行一个表达式
  7. Ctrl + F5 重新启动调试
